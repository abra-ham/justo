### Justo
Hope you like it!

## Client

To install dependencies

`cd client && yarn`

To run

`yarn start-dev`

Open `localhost:9000/items`

## Server
To install dependencies

`cd server && yarn`

To run

`yarn start-dev`

Unit tests can run without running the server just run

`yarn test`

in your terminal

*  Note:
Please use Node 10.0.0 or above.
